# ApiCountriesV1

Aplicación de consola que consume la [REST Countries API](https://restcountries.com) para buscar países y guardarlos en favoritos.

**Autor:** Ramón F. Pérez Álvarez  
**Formación:** FP DAW - IES Son Ferrer

## Cómo ejecutar el proyecto

1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta:

```bash
dotnet run
```

---

**Alternativa:**

```bash
dotnet build
.\bin\Debug\net10.0\ApiCountriesV1.exe
```
